Creates a TransactionModel with the contents and save to local database.

When a TransactionModel is created it is not added to the next block immediately. It needs to wait until the _blkTimer runs again to check if the oldest transaction was created more than _blkInterval duration or if the TransactionModel waiting to be added in a BlockModel already reached 100Kb in size.
