Creates a Uint8List representation of the block.

The serialized BlockModel is created by combining in a Uint8List the BlockModel.header and the block body, that is built from the TransacionModel list by TransactionService.serializeTransactions.

Implementation

Uint8List serialize(Uint8List body) {
  Uint8List head = header();
  return (BytesBuilder()
        ..add(head)
        ..add(body))
      .toBytes();
}
